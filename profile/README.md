## Key Features of Adobe Substance Painter

- **PBR Texturing** – **Adobe Substance Painter** automatically handles metal/roughness or specular/gloss.  
- **Smart Materials** – **Adobe Substance Painter** applies context‑aware, non‑destructive materials.  
- **Smart Masks** – Create procedural dust, edge wear, or leaks inside **Adobe Substance Painter**.  
- **Particle Painting** – Paint scattered elements like gravel or paint splatters using **Adobe Substance Painter**.  
- **Anchor Points** – Reference effects across layers inside **Adobe Substance Painter**.  
- **Real‑Time Viewport** – See PBR lighting and tessellation live in **Adobe Substance Painter**.  
- **Baking Tools** – Bake AO, curvature, ID maps, and thickness with **Adobe Substance Painter**.  
- **Automation** – Use Python scripts and presets inside **Adobe Substance Painter**.  
- **Export Templates** – Output textures for Unity, Unreal, or custom engines via **Adobe Substance Painter**.  
- **Layers System** – Non‑destructive painting with blend modes inside **Adobe Substance Painter**.  
- **Cross‑Platform** – Use **Adobe Substance Painter** on Linux, Web, and Android.  
- **Keyboard Shortcuts** – Navigate **Adobe Substance Painter** faster with hotkeys.  
- **Push Notifications** – Get alerts from **Adobe Substance Painter** when baking completes.  
- **Material Library** – Save custom materials inside **Adobe Substance Painter** for reuse.  
- **Search & Replace** – Find any texture or setting instantly using **Adobe Substance Painter** advanced search.
